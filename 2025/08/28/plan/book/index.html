JVM

一、核心经典（必读）

1. 《深入理解Java虚拟机》（第3版）- 周志明

  • 特点：国内JVM领域权威著作，涵盖内存模型、垃圾回收、字节码执行等核心内容
  • 新增内容：Java 8~12特性（ZGC、Graal编译器等）
  • 适合人群：所有Java开发者，特别是中高级开发者和架构师

2. 《Java虚拟机规范》（Java SE版）- Oracle官方

  • 特点：JVM权威规范文档
  • 内容：Class文件格式、字节码指令集、内存模型等
  • 链接：Oracle官方文档

二、进阶与调优

3. 《Java性能权威指南》- Scott Oaks

  • 特点：从JVM角度分析Java性能优化
  • 内容：垃圾回收、JIT编译、线程管理等实战技巧

4. 《垃圾回收算法与实现》- 中村成洋

  • 特点：系统讲解GC算法
  • 内容：标记-清除、分代、G1、ZGC等算法及伪代码实现

三、底层与扩展

5. 《自己动手写Java虚拟机》- 张秀宏

  • 特点：通过实现简化版JVM理解核心原理
  • 语言：使用Go语言实现

6. 《The Java Virtual Machine: An Interpreted Approach》- Bill Venners

  • 特点：从解释器视角分析JVM设计
  • 重点：字节码执行和JIT优化

四、新特性与未来方向

7. 《Java并发编程实战》- Brian Goetz

  • 相关JVM内容：内存模型、happens-before规则等

8. GraalVM相关资源

  • 推荐：GraalVM官方文档
  • 链接：GraalVM官网

五、免费资源

9. 《JVM Anatomy Quarks》系列博客

  • 作者：Aleksey Shipilëv
  • 特点：短篇技术文章解析JVM细节
  • 链接：OpenJDK Wiki

10. OpenJDK源码

  • 推荐：直接阅读HotSpot VM源码
  • 链接：OpenJDK GitHub

选择建议

  • 入门/面试：优先读《深入理解Java虚拟机》前6章
  • 性能调优：《Java性能权威指南》+《垃圾回收算法与实现》
  • 硬核实践：《自己动手写Java虚拟机》+ OpenJDK源码

Java 语言

一、Java 基础

1. 《Java核心技术 卷I》（原书第12版） - Cay S. Horstmann

  • 特点：全面系统的Java入门教程
  • 内容：基础语法、面向对象、集合框架等
  • 适合：Java初学者

2. 《Effective Java》（第3版） - Joshua Bloch

  • 特点：Java编程最佳实践指南
  • 内容：90条Java编程建议和设计原则
  • 适合：有一定基础的Java开发者

二、Java 进阶

3. 《Java编程思想》（第4版） - Bruce Eckel

  • 特点：深入讲解Java编程思想
  • 内容：面向对象、泛型、并发等高级特性
  • 适合：想深入理解Java的开发者

4. 《Java并发编程实战》 - Brian Goetz

  • 特点：Java并发编程权威指南
  • 内容：线程安全、并发工具类、性能调优
  • 适合：需要开发高并发应用的开发者

三、Java 新特性

5. 《Modern Java in Action》 - Raoul-Gabriel Urma

  • 特点：全面介绍Java 8-17新特性
  • 内容：Lambda、Stream API、模块系统等
  • 适合：想学习现代Java特性的开发者

6. 《Java新特性核心详解》 - 周志明

  • 特点：深入解析Java 9-17新特性
  • 内容：模块化、JShell、var语法等
  • 适合：国内开发者学习新特性

四、Java 实战

7. 《Spring实战》（第6版） - Craig Walls

  • 特点：Spring框架权威指南
  • 内容：Spring Boot、Spring Cloud等
  • 适合：Java企业级开发

8. 《阿里巴巴Java开发手册》

  • 特点：阿里Java开发规范
  • 内容：编码规范、工程结构、安全规约
  • 适合：企业Java开发团队

五、免费资源

9. Oracle官方Java教程

  • 链接：https://docs.oracle.com/javase/tutorial/

10. Java Design Patterns

  • 特点：Java设计模式实例
  • 链接：https://java-design-patterns.com/
